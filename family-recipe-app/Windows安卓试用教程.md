# 家庭菜谱 App - Windows + 安卓手机 试用教程

> **零基础版，每一步都写清楚，不跳任何环节**

---

## 整体流程一览

```
第①步：在电脑上装 Node.js（一次性，5分钟）
第②步：下载项目压缩包并解压（1分钟）
第③步：在电脑上启动 App（2分钟）
第④步：查电脑的 IP 地址（1分钟）
第⑤步：手机浏览器打开（10秒）
```

**手机上不装任何东西、不存任何文件。** 手机只是用浏览器打开一个网址，所有数据都在电脑上。

---

## 第①步：安装 Node.js

Node.js 是运行这个 App 的必备环境，就像玩游戏要装游戏引擎一样。

### 1.1 下载

1. 打开电脑浏览器（Chrome / Edge 都行）
2. 地址栏输入：`https://nodejs.org` 回车
3. 你会看到页面上有两个绿色下载按钮，点左边那个 **"LTS"**（长按支持版本）
   - 版本号大概是 `v20.x.x` 或 `v22.x.x`，都行
4. 等下载完成

### 1.2 安装

1. 找到下载的文件（通常在"下载"文件夹里），文件名类似 `node-v20.xx.x-x64.msi`
2. **双击**它
3. 弹出安装向导，点 **"Next"**
4. 勾选 "I accept the terms..."（我同意），点 **"Next"**
5. 安装路径不用改，点 **"Next"**
6. 后面一路点 **"Next"**，直到看到 **"Install"**，点它
7. 可能弹出"是否允许更改"，点 **"是"**
8. 等进度条走完，点 **"Finish"** 完成

### 1.3 验证装好了

1. 按键盘上 **Win 键**（左下角带 Windows 图标的那个键）
2. 直接输入 `cmd`，你会看到"命令提示符"，**点它**打开
3. 弹出一个黑色窗口
4. 在黑色窗口里输入：
```
node -v
```
5. 按回车，如果显示类似 `v20.11.0` 这样的版本号，说明**装好了**，继续下一步
   - 如果显示"不是内部或外部命令"，说明没装好，重启电脑后再试

---

## 第②步：下载项目并解压

### 2.1 下载

1. 在对话中找到 `family-recipe-app.tar.gz` 文件，下载到电脑
2. 默认会下载到"下载"文件夹

### 2.2 解压

Windows 自带的解压不支持 `.tar.gz` 格式，需要用解压软件。

**如果你电脑有 7-Zip 或 Bandizip 或 WinRAR：**
1. 右键点 `family-recipe-app.tar.gz`
2. 选"解压到当前文件夹"或"提取到 family-recipe-app.tar"
3. 解压后会得到一个 `family-recipe-app.tar` 文件，再右键它解压一次
4. 最终得到 `family-recipe-app` 文件夹

**如果没有解压软件：**
1. 去 https://www.7-zip.org 下载 7-Zip（免费），装好
2. 再按上面步骤解压

### 2.3 把文件夹放到好找的位置

把 `family-recipe-app` 文件夹移到桌面或 D 盘根目录，方便后面操作。
比如放到：`D:\family-recipe-app`

---

## 第③步：启动 App

### 3.1 打开命令提示符

1. 按 **Win 键**，输入 `cmd`，点"命令提示符"
2. 黑色窗口打开

### 3.2 进入项目文件夹

在黑色窗口输入（注意 cd 后面有空格）：
```
cd /d D:\family-recipe-app
```

> ⚠️ 把 `D:\family-recipe-app` 换成你实际放的路径。如果你放桌面了，路径是 `C:\Users\你的用户名\Desktop\family-recipe-app`

按回车。如果没报错，说明进去了。

### 3.3 编译前端（只做一次）

依次输入以下命令，**每行输入后按回车，等上一条跑完再输下一条**：

```
cd web
npm install
```
> 这一步会下载前端依赖，可能等 1-2 分钟，屏幕上会滚很多字，正常现象，等它停下来。

```
npm run build
```
> 这一步编译前端，很快，几秒钟。

```
xcopy /E /I dist ..\server\web-dist
```
> 把编译好的文件复制到后端目录。看到"复制了 XX 个文件"就对了。

### 3.4 安装后端依赖并启动

```
cd ..\server
npm install
```
> 又是下载依赖，等 1-2 分钟。

```
npm start
```

看到这行字就说明**启动成功了**：
```
家庭菜谱服务已启动: http://localhost:3000
```

> ⚠️ **这个黑色窗口不要关！** 关了 App 就停了。可以最小化，但不能关。

### 3.5 先在电脑上验证

1. 打开电脑浏览器
2. 地址栏输入：`http://localhost:3000`
3. 能看到登录页，说明 App 跑起来了，继续下一步

---

## 第④步：查电脑的 IP 地址

手机要访问电脑，需要知道电脑在局域网的 IP 地址。

### 4.1 查 IP

1. **再开一个命令提示符窗口**（之前那个别关，App 在跑着）
   - 按 Win 键，输入 `cmd`，点开
2. 输入：
```
ipconfig
```
3. 按回车，会输出一堆文字
4. 找到 **"无线局域网适配器 WLAN"** 或 **"无线局域网适配器 WLAN 2"** 这一段
5. 在它下面找 **"IPv4 地址"** 那一行
6. 右边那串数字就是你的 IP，类似：`192.168.1.100` 或 `192.168.0.103`

**把这串数字记下来**（抄在纸上或拍照）。

> ⚠️ 不要抄"子网掩码"或"默认网关"，要的是 **IPv4 地址**。
> ⚠️ 如果你电脑是插网线的，找"以太网适配器"下面的 IPv4 地址。

---

## 第⑤步：手机访问

### 5.1 确认同一个 WiFi

1. 看你手机连的 WiFi 名字
2. 看你电脑连的 WiFi 名字
3. **必须是一样的**（比如都连着 "TP-LINK_5G" 这个 WiFi）

### 5.2 手机打开

1. 打开手机浏览器（Chrome / 自带浏览器都行）
2. 地址栏输入：
```
http://你刚才记的IP:3000
```
比如你的 IP 是 `192.168.1.100`，就输入：
```
http://192.168.1.100:3000
```
3. 按"前往"/"搜索"

### 5.3 看到登录页就成功了！

用演示账号登录：
- 用户名：`demo`
- 密码：`123456`

## 如果手机打不开怎么办

### 情况 1：一直转圈或"无法访问"

最可能是 **Windows 防火墙拦截了**。解决：

1. 按 Win 键，输入"防火墙"，点"Windows Defender 防火墙"
2. 左侧点"允许应用通过 Windows Defender 防火墙"
3. 点右下角"更改设置"（需要管理员权限）
4. 在列表里找 **"Node.js"**
5. 把它后面的 **"专用"和"公用"都勾上**
6. 点"确定"
7. 手机重新刷新页面试试

> 如果列表里没有 Node.js，点下方"允许其他应用"，浏览找到 `C:\Program Files\nodejs\node.exe` 添加。

### 情况 2：防火墙关了还是不行

可能是路由器开了"AP隔离"（设备隔离），导致同一 WiFi 下设备互相看不见。

**验证方法**：在电脑浏览器打开 `http://localhost:3000` 能看到，但手机打不开，基本就是这个问题。

**解决办法**：
1. 浏览器打开路由器管理页（通常是 `http://192.168.1.1` 或 `http://192.168.0.1`）
2. 登录（用户名/密码在路由器背面贴着）
3. 找"无线设置"或"高级设置"
4. 找到"AP隔离"或"客户端隔离"或"设备隔离"
5. **关掉它**，保存
6. 手机重新试

### 情况 3：你不在家 / 是公司 WiFi

公司、学校、公共 WiFi 通常会隔离设备，没法互访。
**回家用自己路由器试。**

### 情况 4：实在搞不定 WiFi

那就先在**电脑浏览器**上体验：打开 `http://localhost:3000`，功能完全一样，就是屏幕大点。

或者告诉我，我帮你部署到公网，手机不依赖 WiFi 也能用。

## 日常使用

### 以后每次用怎么启动？

1. 打开 cmd
2. `cd /d D:\family-recipe-app\server`
3. `npm start`

不用再装依赖了，三秒就启动。

### 怎么关闭 App？

直接关掉那个黑色命令提示符窗口就行。

### 数据存在哪？

- 菜谱、用户等数据：`D:\family-recipe-app\server\data\recipe.db`
- 上传的图片视频：`D:\family-recipe-app\server\uploads\`
- 备份就复制这两个东西
