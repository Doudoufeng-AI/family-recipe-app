# 家庭菜谱 App - 试用指南（零基础版）

## 核心原理

```
你的电脑（运行 App 服务）  ←——WiFi——→  你的手机（浏览器打开使用）
    IP: 192.168.x.x                  访问 http://192.168.x.x:3000
```

- 电脑当服务器，手机只用浏览器访问，**手机上不装任何东西、不存任何文件**
- 所有菜谱、照片、视频数据都存在你电脑上

---

## 第一步：确认电脑系统

| 你的电脑 | 操作 |
|---------|------|
| Mac | 打开「终端」App（Command+空格 搜索"终端"） |
| Windows | 打开「命令提示符」（Win键 搜索"cmd"） |

---

## 第二步：安装 Node.js（运行环境）

这个 App 需要一个叫 Node.js 的运行环境，就像玩游戏需要装游戏引擎一样。

### Mac 安装方式（二选一）

**方式 A（推荐）：**
1. 打开浏览器，访问 https://nodejs.org
2. 点击左边绿色的 "LTS" 下载按钮
3. 下载后双击安装，一路点「继续」即可

**方式 B（如果你有 Homebrew）：**
```bash
brew install node
```

### Windows 安装方式

1. 打开浏览器，访问 https://nodejs.org
2. 点击左边绿色的 "LTS" 下载按钮
3. 下载后双击运行，一路点「Next」即可

### 验证安装成功

在终端/命令提示符输入：
```
node -v
```
如果显示类似 `v20.x.x` 或 `v22.x.x`，说明装好了。

---

## 第三步：下载项目并解压

1. 找到 `family-recipe-app.tar.gz` 文件（在对话中下载）
2. 解压到你想放的位置：
   - **Mac**：双击 `.tar.gz` 文件自动解压，得到 `family-recipe-app` 文件夹
   - **Windows**：用 7-Zip 或 Bandizip 解压（Windows 自带的不支持 tar.gz，需装解压软件）

记住解压后的文件夹路径，比如：
- Mac：`/Users/你的用户名/Downloads/family-recipe-app`
- Windows：`C:\Users\你的用户名\Downloads\family-recipe-app`

---

## 第四步：启动 App

### Mac 启动

1. 打开「终端」
2. 输入 `cd `（注意 cd 后面有个空格），然后把 `family-recipe-app` 文件夹**拖到终端窗口里**，会自动填入路径，按回车
3. 输入以下命令并回车：
```bash
./start.sh
```
4. 等待 1-3 分钟（首次需要下载依赖），看到 `家庭菜谱服务已启动: http://localhost:3000` 就成功了

### Windows 启动

Windows 无法直接执行 `.sh` 脚本，按以下步骤：

1. 打开「命令提示符」（按 Win键，输入 cmd 回车）
2. 输入以下命令进入项目目录（把路径换成你的实际路径）：
```cmd
cd C:\Users\你的用户名\Downloads\family-recipe-app
```
3. 先编译前端：
```cmd
cd web
npm install
npm run build
xcopy /E /I dist ..\server\web-dist
cd ..\server
npm install
npm start
```
4. 看到 `家庭菜谱服务已启动: http://localhost:3000` 就成功了

> **注意**：启动后不要关闭终端窗口！关了就等于关掉了服务。

---

## 第五步：找到电脑的 IP 地址

手机需要知道电脑的 IP 地址才能访问。

### Mac 查 IP

1. 点左上角  → 系统设置 → Wi-Fi
2. 点你连接的 WiFi 名字旁边的「详细信息」
3. 找到「IP 地址」，类似 `192.168.1.100`

或者终端输入：
```bash
ipconfig getifaddr en0
```

### Windows 查 IP

1. 打开命令提示符，输入：
```cmd
ipconfig
```
2. 找到「无线局域网适配器 WLAN」下面的「IPv4 地址」，类似 `192.168.1.100`

---

## 第六步：手机访问

1. 确保手机和电脑连着**同一个 WiFi**（同一个 WiFi 名字）
2. 打开手机浏览器（Safari / Chrome 都行）
3. 地址栏输入：`http://你的电脑IP:3000`
   - 比如：`http://192.168.1.100:3000`
4. 就能看到 App 了！

> 也可以把这个链接发到微信，在微信里直接点开使用。

## 常见问题

### Q：手机打不开页面？
可能原因和解决办法：

1. **电脑防火墙拦截了**
   - Mac：系统设置 → 网络 → 防火墙 → 关闭，或允许 Node.js 入站
   - Windows：控制面板 → Windows Defender 防火墙 → 允许应用通过防火墙 → 找到 Node.js 勾选

2. **不在同一个 WiFi**
   - 确认手机和电脑连的是同一个 WiFi 名字
   - 公司/学校 WiFi 可能隔离设备，回家用自己路由器试试

3. **IP 地址搞错了**
   - 重新查一遍 IP，确保是 `192.168.x.x` 格式
   - 电脑端先自己测试：浏览器打开 `http://localhost:3000`，能打开说明服务正常

4. **端口被占用**
   - 终端报错 `EADDRINUSE`，说明 3000 端口被占
   - 换个端口：`PORT=3001 node src/app.js`（Mac）或 `set PORT=3001 && node src/app.js`（Windows），手机访问时端口也改成 3001

### Q：关闭终端后 App 就不能用了？
是的，终端窗口关了服务就停了。想长期运行：
- Mac：终端输入 `nohup node src/app.js &`
- Windows：暂时没有简单方案，保持窗口开着即可

### Q：数据存在哪？
所有数据存在 `family-recipe-app/server/data/recipe.db`（菜谱、用户等）和 `family-recipe-app/server/uploads/`（图片视频）。备份这两个目录就行。

### Q：每次用都要启动吗？
是的，每次重启电脑后需要重新启动服务。但不用重新安装依赖，直接 `cd server && npm start` 即可。
